⚙️ How It Works

🧠

Logistic Regression

Computes a weighted sum of your inputs, passes it through a sigmoid function to output a dropout probability.

📐

Feature Scaling

Each input is standardised (mean=0, std=1) before prediction so no single feature dominates the model.

⚡

No Server Needed

The model weights are embedded directly in JavaScript. Works offline, instantly, just by opening this file.

🎯

~88% Accuracy

Trained on 500 synthetic student records. Attendance, GPA and prior score are the strongest predictors.
